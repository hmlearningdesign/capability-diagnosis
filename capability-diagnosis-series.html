Four practice groups, four capability gaps

Alderney & Associates is a mid-size professional services firm. Four practice groups have each raised a performance problem. The stakeholder in each case has a theory about the cause. Your job is to test it against the evidence, not assume it's right. Not all the evidence points the right way, some of it is a plausible distraction.

Work through each case in any order. Every case runs the same way: read the brief, review the evidence, classify the root cause, defend the classification with the strongest evidence, recommend an intervention, then view the resulting capability gap summary.

All four cases complete. View the combined exhibit showing how the same diagnostic framework produced four different, evidence-based conclusions.
